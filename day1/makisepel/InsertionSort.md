## 삽입 정렬 (Insertion Sort)

탐색 범위가 정해진 배열에서, 원소를 하나씩 꺼내 이미 정렬된 부분에서 알맞은 위치에 삽입하며 정렬하는 방법

- 장점
    - 구현이 매우 간단하고 Stable하다
    - 배열이 정렬되어 있을 수록 시간 복잡도가 낮다
- 단점
    - 데이터가 많을수록 성능이 급격히 저하된다

---

```c
for (i = 1; i < n; i++) {

    key = arr[i];
    j = i - 1;

    while (j >= 0 && arr[j] > key) {
        arr[j + 1] = arr[j];
        j--;
    }

    arr[j + 1] = key;
}
```

시간 복잡도 : O(n^2) → 정렬되어 있을 경우 O(n)
