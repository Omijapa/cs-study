## 선택 정렬 (Selection Sort)

탐색 범위가 정해진 배열에서, 배열의 정해진 위치에 들어갈 원소를 순차적(오름차순 혹은 내림차순)으로 탐색하여 해당 위치에 맞는 최솟값(또는 최댓값)을 선택해 교환하며 정렬하는 방법

- 장점
    - 구현이 간단하고 Swap 함수 호출 횟수가 적다
- 단점
    - 비교 횟수가 많아 비효율적이고 이미 정렬된 경우에도 동일한 비교를 수행한다

---

```c
for (i = 0; i < n - 1; i++) {
    min = i;

    for (j = i + 1; j < n; j++) {
        if (arr[j] < arr[min])
            min = j;
    }

    swap(arr[i], arr[min]);
}
```

시간 복잡도 : O(n^2)
